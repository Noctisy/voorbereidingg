Er is een populate.php bestand, het link is http://localhost/examenvooruit/populate.php
als je op het pagina gaat en op register drukt word een admin account aangemaakt met de volgende gegevens
username: admin
wachtwoord: admin

Om het database te gaan populaten met gegevens voor de volgende tabellen: artikel, leverancier, locatie, voorraad. het bestand heet: sql_populate.sql
je moet jammer genoeg gaan kopieren en naar de database hengelsport gaan. In sql plakken en go drukken. deze gegevens kunnen niet verwijderd worden via de CRUD.
Als je gegevens wilt verwijderen via de CRUD maak eerst nieuwe gegevens aan, en dan kun je die met gemakt verwijderen. Je kunt alle gegevens wel aanpassen.

als je admin bent word je door gestuurt naar homepagina.php.
als je een user bent dan word je door gestuurt naar welcome_user.php waar je bijna alle paginas hebt behalve nieuwe medewerkers toevoegen

als je de view van een klant wilt zien moet je uitloggen om naar index.php te komen. Vanuit daar kun je zien hoe de klant dingen ziet

missende functies: notificatie wanneer voorraad laag is
voorraad overzicht is niet bijna af. Kan alleen niet aanpassen en verijderen want tijd tekort
